# cf-nodejs2
A very simple nodejs example to push quickly to Cloud Foundry

## To run locally

You need `npm` installed on your machine.

```
	$> npm install
	$> node server.js
```


## Push to cloud foundry

You need `cf cli` installed on your machine.

```
	$> cf push
	$> cf apps
	$> curl http://cf-nodejs2.apps.<domain>.io
```
